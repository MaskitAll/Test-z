Задача 14. "Простое тестовое"

Условие: Это реальное тестовое задание для начинающего верстальщика. Данный word-документ перевести в html страницу. CSS использовать не нужно. Требования: Аккуратный, чистый html код; Валидность; Все логические элементы выделяем семантически правильными тегами. Ссылки должны быть - ссылками (<a>), списки - списками (<ul>, <ol>) и тд; Текст выделенный желтым цветом, обернуть как цитату; Заголовки использовать в соответствии с их уровнем (<h1..h6>).
