Условие: Создайте html-файл, результат которого показан на рисунке. Используйте теги html для оформления в виде таблицы. Обратите внимание, что в таблице присутствуют текстовые ссылки и ссылки-изображения. Изображение статьи можно скачать здесь. Кстати, последнюю строку на картинке тоже необходимо повторить.
Используются теги <table>, <thead>, <tr>, <td> для создния таблицы. Тег <a> для создания ссылки
